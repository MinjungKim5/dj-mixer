# 💿 ㅋㅁㅇ — 유튜브 DJ 믹서

## 배경

'슬픔의 케이팝 파티'를 아시나요?
코로나 시절 즈음에 우리나라 디제이들이 게릴라성으로 열었던 케이팝 파티였는데요,
제가 속해있는 소모임 케이팝 문화 연구회에서도 매주 화요일 점심마다 작은 케이팝 파티를 열고 있습니다.

파티에는 언제나 디제이가 필요하고 케문연에서는 모임장인 정민지님이 주로 디제이를 맡아주고 계십니다.
하지만 노션 페이지에 신청 영상들을 올리고 한 곡이 끝날때마다 노션페이지로 돌아가 링크를 클릭하고 새창에서 다음 곡을 트는 과정이 이 파티에 마찰요소가 된다는 생각이 들었습니다.

디제이의 본분은 파티에 음악이 끊기지 않게 하는것. ㅋㅁㅇ은 유튜브 영상을 소스로 하는 파티에서 사용할 수 있는 미니멀한 앱을 지향하며 만들었습니다.

## 해결하고 싶었던 문제

| 기존 방식                            | ㅋㅁㅇ                                        |
| ------------------------------------ | --------------------------------------------- |
| 노션에서 링크를 하나씩 클릭          | 재생목록 URL 하나로 큐에 일괄 등록            |
| 곡이 끝나면 수동으로 다음 곡 재생    | 자동 전환 + 크로스페이드                      |
| 곡 전환 시 음악이 끊김               | 더블덱으로 다음 곡을 미리 준비, 페이드인/아웃 |
| 프로젝터에 띄우려면 브라우저 탭 공유 | 프로젝션 전용 창으로 깔끔한 외부 출력         |

## 핵심 기능

### 더블 덱

두 개의 독립된 YouTube 플레이어(Deck A / Deck B)로 한 쪽이 재생되는 동안 다른 쪽에 다음 곡을 준비할 수 있습니다.

### 스마트 큐-인

YouTube 재생목록 URL을 넣으면 A/B 큐에 한 곡씩 교대로 자동 분배합니다.
단일 곡은 곡이 적은 큐에 자동으로 추가됩니다.

### 크로스페이더 믹싱

두 덱 사이의 볼륨 비율을 실시간으로 조절하여 자연스러운 곡 전환이 가능합니다.
명령어 입력(`l30`, `r10`, `m`)으로 시간 지정 자동 페이드도 지원합니다.

### 자동 재생 모드

켜두면 곡이 끝나기 전에 자동으로 다음 덱을 시작하고, 크로스페이드를 실행하며, 프로젝션까지 전환합니다. DJ가 손을 놓아도 파티가 멈추지 않습니다.

### 프로젝션 출력

별도 창으로 현재 재생 중인 덱의 영상을 전체화면 출력합니다. 프로젝터나 외부 모니터에 띄울 수 있습니다.

### 키보드 단축키

한글 모드/CapsLock과 무관하게 물리 키 위치 기반으로 동작합니다.

| 키                | 동작                 |
| ----------------- | -------------------- |
| `Q`               | 스마트 큐-인 모달    |
| `A` / `B`         | Deck A / B 재생 토글 |
| `U` `I` / `O` `P` | Deck A / B 배속 조절 |

## 기술 스택

- **Next.js 16** — App Router, 정적 빌드(CDN 배포 가능)
- **Tailwind CSS 4** — 다크/라이트 테마
- **YouTube IFrame API** — 재생 제어
- **YouTube Data API v3** — 메타데이터, 재생목록 조회
- **BroadcastChannel API** — 프로젝션 창과의 통신
- 서버/DB 없음, 100% 클라이언트 사이드
